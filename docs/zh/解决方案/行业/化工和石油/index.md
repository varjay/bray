Flow-Tek™ M1™ 和 M4™ 系列恶劣工况金属阀座式球阀在最具挑战性的流体控制条件下提供卓越的性能和可靠性。M1™ 和 M4™ 系列隔离阀确保在要求绝对安全性和可靠性的高温、高压、腐蚀性和侵蚀性应用、关键型应用以及杀伤性工况应用中实现零泄漏。

案例分析：
丙烯生产工艺借助 Tri Lok® 三偏心蝶 阀 提升了安全性和效率
在丙烯生产工艺中，热空气和丙烷被注入催化剂床，从而获得丙烯附产品。热空气来自于燃气轮 机，被注入加热器升压器，以将温度 提高到所需的 1100°F （593°C）。当一个或多个燃气轮机停 机进行维护时，压缩机将用于维持注射加热器中的压力。压缩机隔离阀是该工艺的一个关键元 素，用于防止燃气轮机运行时压缩机出现背压。
